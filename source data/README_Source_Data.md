# Source data

This folder contains the source data tables used to generate the figures for the manuscript:

**Microplastic-induced growth reprogramming decouples soil cadmium availability from phytoextraction in *Solanum nigrum***

## Data availability

The raw sequencing data generated in this study have been deposited in the NCBI Sequence Read Archive (SRA) under BioProject accession **PRJNA1398028**. These include the metagenomic and transcriptomic sequencing reads used for the microbiome and RNA-seq analyses.

Because the sequencing datasets are large and are already available through NCBI SRA, they are not duplicated here. This folder instead provides the figure source data tables and selected derived analysis outputs used to generate the plotted results reported in the manuscript.

## Notes on source data interpretation

The CSV files provide the source values used to generate the corresponding figure panels. They are intended as **figure source data**, not as a complete raw experimental database.

Several figure panels were generated from high-dimensional sequencing datasets, including metagenomic profiles, transcriptomic expression matrices, pathway-level enrichment results, biomarker tables, and integrated correlation or modelling outputs. The raw sequencing reads are available from NCBI SRA under **PRJNA1398028**.

For panels involving derived variables, such as cadmium extraction, biomass allocation, translocation factors, log-response ratios, contribution estimates, decoupling indices, PCA-derived indices, or regression-based path coefficients, the values in the source data files represent the plotted or modelled source values used in the corresponding analyses. When variables were calculated from multiple measurements or treatment-level summaries, the derived values are provided directly as figure source values. 
## Figure-specific notes

### Figure 1

`Source_Data_Fig1.csv` contains log-response ratios of total Cd extraction and the relative contributions of biomass and Cd concentration components. These values were used to generate Fig. 1a and Fig. 1b.

### Figure 2

`Source_Data_Fig2.csv` contains the plotted source values for biomass, biomass allocation, Cd concentration, translocation factor, and Cd extraction traits. Some Cd extraction and allocation-related variables are derived values used for plotting and statistical comparison.

### Figure 3

`Source_Data_Fig3.csv` contains rhizosphere physicochemical variables used for the effect-size heatmap and related soil-trait analyses. Microbial community, KEGG functional profiles, LEfSe biomarkers, and correlation-based summaries were generated from processed metagenomic outputs derived from the SRA-deposited sequencing data.

### Supplementary Figures S2-S6

`Source_Data_FigS2-S6.csv` contains the plotted source values for the screening experiment, including biomass, Cd concentration, translocation, Cd extraction, root traits, photosynthesis-related traits, and representative correlation analyses.

### Supplementary Figure S7

`Source_Data_FigS7.csv` contains nutrient concentration data for berry, shoot, and root tissues under CK, PE0.5, and PVC5 treatments.

### Figure 5

Figure 5 was generated from integrated analysis outputs combining plant traits, rhizosphere variables, metagenomic biomarkers, transcriptome-derived pathway scores, PCA-derived indices, and regression-based models. The values provided here represent processed or derived source values supporting the plotted panels and model summaries.
